Disables most things printed to console by Gearbox, leaving a nice clean console for logging.

Disabled Messages are:
Debug Messages
Item Messages
Weapon Messages
Ammo Messages
Credits Messages
Challenge Messages
Mission Messages
Fast Travel Station Messages
Level Up Messages
Join/Leave Game Messages
Change Name Messages
